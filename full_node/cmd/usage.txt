Instruction on Usage
1. Add mutual TCP connection to Peer
$ add_peer PEER_IPV4 PEER_PORT

2. List all current peers
$ list_peer

3. Start Mining ad tail
$ start

4. Stop Mining
$ stop

5. Restart Mining at new tail
$ restart

6. Show Blockchain last ${NUMBER} layers
$ show NUMBER

7. Show Public Key in Hex
$ key

8. Show network topology
$ network

9. Get peers of given endpoint.
$ introduce PEER_IPV4 PEER_PORT
